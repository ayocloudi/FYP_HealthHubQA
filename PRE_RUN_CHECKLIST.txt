TROUBLESHOOTING DURING RUN
==========================

If you see:
  "ModuleNotFoundError: No module named 'X'"
  → pip install X
  → Try running again

If you see:
  "RepositoryNotFoundError" for Hugging Face models
  → Check internet connection
  → Try again (download might timeout)

If you see:
  "⚠ Skipping GPT: missing OPENAI_API_KEY"
  → This is expected if you didn't set the key
  → Only Flan-T5 conditions will run
  → If you want GPT, set: $env:OPENAI_API_KEY = "key"
  → Re-run the script

If script takes 1+ hour and not done:
  → Normal for first run (downloads ~5GB models)
  → Grab coffee ☕
  → Second run will be much faster

If you get "CUDA out of memory":
  → Script already defaults to CPU
  → If forcing GPU, switch to CPU: DEVICE = "cpu"
