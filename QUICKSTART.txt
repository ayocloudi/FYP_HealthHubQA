TROUBLESHOOTING
===============

Q: "No module named 'openai'" when running with GPT
A: pip install openai

Q: "⚠ Skipping GPT: missing OPENAI_API_KEY"
A: Set the environment variable first (see OPTION 2 above)

Q: Script times out / very slow
A: Normal on CPU. First run downloads models (~5GB).
   Subsequent runs use cached models (much faster).
   Grab coffee ☕ and wait.

Q: CUDA out of memory
A: Already set to CPU by default. If you have GPU:
   Edit line 38: DEVICE = "cuda"

Q: API rate limit error
A: Slow down by adding delay between calls.
   Or reduce SAMPLE_SIZE (line 33: SAMPLE_SIZE = 50)

Q: "RepositoryNotFoundError" for Hugging Face models
A: Check internet connection, try again.
